Input de tipo radio

Cada radio del grupo debe tener el mismo valor de name.
El navegador crea un array con los elementos que comparte el mismo name.
Hay propiedades que se asignan a todo el grupo y propiedades para cada elemento.

Para obtener la opción chequeada, se accede a cada elemento del array y a su propiedad checked.

Selecciona tu actor favorito: 
window.onload = () => {

    let btn_consultar = document.getElementById('consultar');
    btn_consultar.onclick = () => {
    
        document.formulario.actores.forEach(actor => {
            if(actor.checked){
                alert(actor.value);
            }
        });
    
    }
    
}
